</br>

## Instructions

</br>

### Install Dependencies
```
$ npm install
```

### Run script
```
$ npm start
```
* The output file will be located on `/data/monthlyPrices.csv`

</br>

## Front-end Instructions

</br>

### Install Dependencies
```
$ npm install
```

### Run on Development Mode
```
$ npm run dev
```

### Build Project
```
npm run build
```
